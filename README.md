# 🌟 Rama Vamsha — The Complete Sacred Lineage

## 📖 Description
**Rama Vamsha** is an immersive, visually rich web experience that presents the complete lineage of Lord Rama — from the divine origins of Vishnu to the ideal king Rama of the Suryavansha.

This project blends mythology, storytelling, and modern web design into a timeline-based journey through generations of sages, kings, and divine beings. It serves as an interactive digital narrative that educates and engages users through animations, visuals, and ambient audio.

---

## ✨ Features
- 🌌 Animated cosmic backgrounds (starfield, mandala, particles)
- 🕉️ Spiritual UI/UX inspired by Indian aesthetics
- 📜 Timeline-based lineage visualization
- 🎧 Built-in ambient music player (Web Audio API)
- 🖼️ Interactive cards for each ancestor
- 📱 Fully responsive (mobile + desktop)
- ⚡ Smooth scrolling and transitions

---

## 🧱 Tech Stack
- HTML5  
- CSS3 (animations, gradients, effects)  
- JavaScript (Vanilla)  
- Web Audio API
